# ctfd-cli
CTFd cli client

## Build

This project requires Go 1.23.3 or later.

To build the project, just run `go build` in the root directory of the project.

## Usage

```
Usage of ./ctfd-cli:
  -baseurl string
    	Base URL for API requests
  -log
    	Log to file
```
